# ChronoPrompt

给 AI 聊天消息自动打上时间戳的浏览器脚本。每次你发送消息时，自动在末尾追加当前时间，方便追踪对话时间线。

目前仅支持 **DeepSeek 网页版**（chat.deepseek.com），浏览器端使用。移动端 App 不支持（App 不经过浏览器脚本）。后续陆续覆盖更多平台。

## 效果

发送「今天天气怎么样」→ 实际发送：

```
今天天气怎么样
2026-05-05 02:27:00
```

## 安装教程（零基础也能看懂）

### 第一步：安装脚本管理器

脚本需要运行在浏览器里，先装一个「脚本管理器」。推荐以下任选一个：

| 管理器 | 推荐理由 | 下载地址 |
|--------|----------|----------|
| **篡改猴 (Tampermonkey)** | 用户最多，最稳定 | [tampermonkey.net](https://www.tampermonkey.net/) |
| **脚本猫 (ScriptCat)** | 国产，国内网络友好 | [scriptcat.org](https://scriptcat.org/) |

> 两个功能几乎一样，选哪个都行。如果你在国内且打不开 Tampermonkey 官网，就装脚本猫。

**安装方式**：打开上面的网站，点「安装」或「下载」，浏览器会提示添加扩展，点「添加扩展」即可。

装好后浏览器右上角会出现一个猴子/猫的图标。

### 第二步：安装脚本

有两种方式，任选一种：

**方式 A：一键安装（推荐）**

1. 打开 [Greasy Fork 脚本页面](https://greasyfork.org/zh-CN)（搜索 `ChronoPrompt`）
2. 点绿色的「安装此脚本」按钮
3. 弹出脚本管理器确认窗口，点「安装」
4. 完成，以后脚本有更新会自动提示升级

**方式 B：手动复制**

1. 打开本仓库里的 `deepseek-timestamp.user.js` 文件
2. 复制全部内容（Ctrl+A → Ctrl+C）
3. 点浏览器右上角的脚本管理器图标 → 选择「创建新脚本」或「添加新脚本」
4. 删除编辑器里的默认内容，把刚才复制的代码粘贴进去
5. 按 Ctrl+S 保存
6. 完成

### 第三步：使用

1. 在电脑浏览器打开 [chat.deepseek.com](https://chat.deepseek.com/)
2. 随便发一条消息
3. **刷新页面**，就能看到你刚发的消息末尾多了时间戳

> ⚠️ **重要提示**：
> 
> 刚发送时，网页上的消息气泡里暂时看不到时间戳。这是因为脚本在发送环节悄悄改了内容，但页面本地渲染用的是旧缓存。**刷新页面后就恢复正常显示。**
> 
> 你用手机打开 DeepSeek 网页版或 App 查看同一条对话，时间戳是能直接看到的——因为服务器存的就是带时间戳的版本，只是网页端需要刷新一下才能拉回来。
> 
> 另外：本脚本只作用于**电脑浏览器**，手机上无法使用。

### 额外说明

- 时间戳格式：`2026-05-05 02:27:00`（年-月-日 时:分:秒）
- 使用你的系统本地时区
- 不收集任何隐私数据，所有代码都在你的浏览器本地运行
- 随时可在脚本管理器中禁用或删除
